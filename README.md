## Deskripsi
Program ini menggunakan konsep Task Parallelism dengan multiprocessing di Python.

## Task yang dijalankan
1. Menjumlahkan digit NRP
2. Mencari nilai maksimum
3. Menghitung faktorial digit terakhir
4. Mengurutkan digit NRP
5. Menghitung rata-rata digit NRP

## Penjelasan
Setiap task dijalankan oleh proses yang berbeda secara paralel.
Hal ini ditunjukkan dengan Process ID (PID) yang berbeda.
